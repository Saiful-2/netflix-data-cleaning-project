# Netflix Data Cleaning Project

## Objective
The objective of this project is to clean and prepare the Netflix Movies and TV Shows dataset for further data analysis.

## Dataset
Netflix Movies and TV Shows Dataset  
Source: Kaggle

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Data Cleaning Steps
1. Loaded dataset using Pandas
2. Inspected dataset structure
3. Removed duplicate rows
4. Handled missing values
5. Converted date columns into datetime format
6. Cleaned text columns
7. Exported cleaned dataset

## Output
A cleaned dataset ready for analysis.
